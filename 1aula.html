Gestão de projetos

A gestão de projetos ajuda as equipes a organizar, acompanhar e executar o trabalho circunscrito a um projeto específico. Podemos considerar um projeto como uma coleção de tarefas necessárias para realizar uma determinada meta. A gestão de projetos permite às equipes planejar, gerir e realizar o trabalho de modo a cumprir com os requisitos do projeto de acordo com o prazo estipulado.

Git & Github

O Git é o sistema de controle de versão, ele é usado para controlar o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software. As vantagens é que ele possui alguns recursos como ter uma área de staging, branches, merges e pull requests.
O github é uma plataforma usada para gerenciar, hospedar códigos e arquivos, além de facilitar a criação de ambientes de colaboração entre pessoas desenvolvedoras.

Função do Docker

Facilita o processo de criação e construção de containers e poder rodar com menor gasto de recursos, além de funcionalidades de segurança.
